【项目概述】

本项目是一个基于 Python + MCP (Model Context Protocol) 的 VC/PE
行业信息自动化处理系统，能够自动从邮箱提取 PitchBook 邮件，
利用 Web 爬虫获取相关报告，并通过 AI 分析生成中文行业周报。

【核心功能】

1. 智能邮件获取系统
   - 采用 MCP 协议连接 QQ 邮箱 IMAP 服务器
   - 自动搜索和筛选 PitchBook 邮件
   - 智能日期过滤（只处理最近 7 天的邮件）
   - 支持大邮件分批获取（10万+字符）

2. Web 反检测爬虫系统
   - 基于 Playwright + Stealth Mode
   - 三层日期过滤机制（邮件日期、链接日期、网页日期）
   - 随机 User-Agent 轮换和真实浏览器行为模拟
   - 智能错误处理（403/429/CAPTCHA）

3. VC/PE 内容分析引擎
   - 自动识别内容类型（交易、融资、市场趋势等）
   - 提取关键主题和标签
   - 按行业、地区、交易阶段分类
   - 生成市场情绪概览

4. 自动报告生成系统
   - 基于 python-docx 生成 Word 周报
   - 完整的中文分析报告

【技术亮点】

✅ MCP 协议应用：通过 Node.js 中间层统一处理邮件协议
✅ 三层日期过滤：确保只处理时效性内容
✅ 反检测爬虫：Stealth Mode + 真实行为模拟
✅ 可配置全量爬取：支持无限制爬取所有符合条件的内容
✅ 完整闭环：邮件 → 爬取 → 分析 → 报告生成

【技术栈】

- Python 3.11+ (主程序)
- Node.js + TypeScript (MCP 邮件服务器)
- Playwright 1.58+ (反检测爬虫)
- MCP (Model Context Protocol)
- IMAP (邮件协议)
- python-docx (报告生成)

【快速运行】

1. 解压附件
2. 创建虚拟环境：python -m venv .venv
3. 激活环境：.venv\Scripts\activate
4. 安装依赖：pip install -r requirements.txt
5. 安装浏览器：playwright install chromium
6. 配置邮箱（编辑 email_credentials.py）
7. 运行程序：python main.py

详细说明请查看项目中的 README.md 文件。
