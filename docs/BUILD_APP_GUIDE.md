# VC/PE PitchBook 系统 - 应用打包指南

## 快速打包

### 方式 1: 完整打包（推荐）

```batch
build.bat
```

这将创建一个包含所有依赖的独立应用。

### 方式 2: 单文件打包

```batch
build_app.bat
```

这将创建一个单一的 .exe 文件（首次启动稍慢）。

## 输出位置

打包完成后，可执行文件位于:
```
dist\VC_PE_PitchBook.exe
```

## 使用说明

1. **复制文件**: 将 `dist` 目录中的所有内容复制到目标位置
2. **首次运行**: 双击 `VC_PE_PitchBook.exe` 启动应用
3. **自动创建**: 程序会自动创建必要的子目录（data、数据储存 等）

## 文件大小

- **完整版**: 约 50-80 MB
- **单文件版**: 约 60-100 MB

## 系统要求

- Windows 10/11
- 无需安装 Python
- 需要网络连接（用于邮件和爬虫功能）

## 注意事项

1. **杀毒软件**: 首次运行可能被杀毒软件拦截，需要添加信任
2. **Playwright**: 爬虫功能需要 Playwright 浏览器，首次使用会自动下载
3. **配置文件**: 配置保存在程序目录的 `data/gui_config.json`

## 分发

打包后的应用可以独立分发，接收者只需:
1. 解压所有文件
2. 双击运行

## 高级配置

### 修改图标

1. 准备一个 `.ico` 文件
2. 编辑 `build_exe.spec`，设置 `icon='your_icon.ico'`
3. 重新打包

### 修改版本信息

编辑 `build_exe.spec` 文件中的版本信息。

## 故障排除

### 打包失败

```batch
# 确保 PyInstaller 已安装
pip install pyinstaller

# 清理后重试
rmdir /s /q build dist
build.bat
```

### 运行时错误

1. 检查是否复制了所有文件
2. 查看程序目录的日志文件
3. 确保网络连接正常
